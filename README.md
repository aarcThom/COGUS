# COGUS
A robot for my son. A lesson in C for me.

COMPONENTS NEEDED FOR PROTOTYPING:

Raspberry Pi Pico 2 H

Micro-USB Data Cable (Data Sync capable)

Breadboard (830 Point)

Jumper Wire Kit (M/M, M/F, and F/F)

DFPlayer Mini MP3 Player

microSD Card (32GB or less)

Speaker (3 Watt, 4 Ohm)

RC522 RFID Module

Rotary Encoder (KY-040)

Potentiometer (10k Ohm, Linear)

Push Buttons (Tactile, 6x6x5mm)

0.96" OLED Display (SSD1306, I2C version)
